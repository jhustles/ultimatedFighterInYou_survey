    About

    For our passion project, we wanted to focus building a machine learning model that can predict a fighter's binary classification: Striker or Grappler. Our project included three components:

    I. Data Preparation

    Our data sources were Kaggle UFC Fight Data that includes all fighter statistics from January 2014 through October 2018, and we webscarped the UFC Fighter Statistic section. After extraction, we transformed the data using MS Excel, Python and Pandas.


    II. Data Analysis

    Next, we performed statistical analysis and create custom ratios to classify 1,100 fighters. Using the output on 3 different machine learning models, we trained and evaluated each model using Sci-Kit Learn. The three models we analyzed were: linear and logistic regression, and decision tree model. We chose the logistic regression model because it gave us 85% accuracy on both training and test data.

    III. Visualization

    Lastly, after determining the best used HTML5, CSS, BootStrap4, JavaScript (ES6), Survey.js, a Pickle file, and Flask Application to help bring our analyses to life and showcase our final product, where you can see find the "Fighter In You". Stay tuned.
